# MVP scope

## Locked decisions

- Global app
- English UI
- Hotel breakfast first
- Bacon is not divided into types
- Status is Yes / No / Not sure
- Map is included
- Photo evidence is included
- Manual hotel creation is included
- No paid APIs

## MVP user flow

1. Search or add hotel manually
2. Create/select hotel
3. Report bacon status
4. See hotel status page
5. See reported hotels on map

## Not in first build

- Restaurant support
- Booking integrations
- Social feed
- Advanced badges
- Public user profiles
- Paid place APIs
- Advanced moderation dashboard
